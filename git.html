Introducción

HTML es el lenguaje de marcado estándar para crear páginas web.

Encabezados

Los encabezados se utilizan para crear los títulos de una página web.

Ejemplo:

<h1>Encabezado h1</h1>

<h2>Encabezado h2</h2>

<h3>Encabezado h3</h3>

<h4>Encabezado h4</h4>

<h5>Encabezado h5</h5>
<h6>Encabezado h6</h6>

Resultado:

Encabezado h1

Encabezado h2

Encabezado h3

Encabezado h4

Encabezado h5
Encabezado h6
