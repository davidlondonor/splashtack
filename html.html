HTML es el lenguaje de marcado estándar para crear páginas web.

Encabezados

Los encabezados se utilizan para crear los títulos de una página web.

Ejemplo:

<h1>Encabezado h1</h1>

<h2>Encabezado h2</h2>

<h3>Encabezado h3</h3>

<h4>Encabezado h4</h4>

<h5>Encabezado h5</h5>
<h6>Encabezado h6</h6>

Resultado:

Encabezado h1

Encabezado h2

Encabezado h3

Encabezado h6

Párrafos

Un párrafo siempre empieza en una línea nueva y usualmente es un bloque de texto.

Ejemplo:

<p>Este es un párrafo</p>

<p>Este es otro párrafo</p>

Resultado:

Este es un párrafo

Este es otro párrafo

Estilos

El atributo style se utiliza para añadir estilos a un elemento: El color, el tipo de fuente, el tamaño del texto y más.

Ejemplo:

<p>Soy un párrafo normal</p>

<p style="color: red">Soy un párrafo rojo</p>

<p style="font-size: 50px">Soy un párrafo grande</p>

Resultado:

CSS

CSS significa hojas de estilo en cascada, estas hojas de estilos controlan el diseño de todas las páginas web.

Con CSS puedes controlar el color, la fuente, el tamaño del texto, el espaciado entre los elementos, la posición de los elementos, las imágenes de fondo, los colores de fondo, y todo lo relacionado con el aspecto y el diseño de una página web.

Ejemplo:

<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>CSS</title>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <div class="elemento1">Elemento 1</div>
        <div class="elemento2">Elemento 2</div>
        <div class="elemento3">Elemento 3</div>
        <div class="elemento4">Elemento 4</div>
        <div class="elemento4">Elemento 5</div>
    </body>
</html>
.elemento1{
    color: red;
}

.elemento2{
    color: #fff;
    background: blue;
    padding: 10px;
    border-radius: 5px;
    width: 150px;
}

.elemento3{
    color: violet;
}

.elemento4{
    font-size: 50px;
}

.elemento5{
    text-decoration: line-through;
}

Rutas de archivos

Una ruta de archivo describe la ubicación de un archivo en la estructura de carpetas de un sitio web.

Ejemplo:

<img src="imagen.jpg">
El archivo "imagen.jpg" se encuentra en la misma carpeta que la página actual.

<img src="imagenes/imagen.jpg">
El archivo "imagen.jpg" se encuentra la carpeta de imágenes.

<img src="/imagenes/imagen.jpg">
El archivo "imagen.jpg" se encuentra la carpeta de imágenes en la raíz de la página web actual.

<img src="../imagen.jpg">
El archivo "imagen.jpg" se encuentra en la carpeta un nivel más arriba de la carpeta actual.

Es una buena práctica utilizar rutas relativas (si es posible).

Cuando utilices rutas relativas, tus páginas web no estarán vinculadas a la URL base actual. Todos los enlaces funcionarán en tu computadora (localhost) y también en tu dominio público futuro.

Etiqueta Head

La etiqueta <head> es el contenedor de los elementos: <title>, <style>, <meta>, <link>, <script>, y <base>.

El elemento <head> es un contenedor de metadatos y se coloca entre la etiqueta <html> y la etiqueta <body>.

Los metadatos definen el título del documento, el conjunto de caracteres, los estilos, los scripts y otra metainformación.

Ejemplo:

Define el juego de carácteres que utilizaras en este caso será la codificación UTF-8:

<meta charset="UTF-8">

Define las palabras claves para los motores de búsqueda:
<meta name="keywords" content="HTML, CSS, JavaScript">

Define una descripción para tu página web.

<meta name="description" content="Free Web tutorials">

Define el autor de una página web:

<meta name="author" content="John Doe">

Elementos

Los sitios web suelen mostrar contenido en varias columnas (como una revista o un periódico).

<header>: Se utiliza para crear un encabezado para un documento o una sección.

<nav>: Se utiliza para crear un conjunto de enlaces de navegación.

<section>: Crea una sección en un documento.

<article>: Crea un contenido autónomo e independiente.

<aside>: Crea contenido aparte del contenido (como una barra lateral).

<footer>: Crea un pie de página para un documento o una sección.

<details>: Te permite crear detalles adicionales que el usuario puede abrir y cerrar bajo demanda.

<summary>: Te permite crear un encabezado para el elemento <details>.

Técnicas de Diseño

Existen cuatro técnicas para crear diseños:

  • Usando un Framework:

    • Puedes utilizar boostrap para crear tus diseños facilmente.

  • Usando la propiedad float:

    • Es común hacer diseños utilizando la propiedad float de css. Los elementos flotantes están vinculados al flujo de los documentos, que puede dañar la flexibilidad.

  • Usando flexbox:

    • El uso de flexbox asegura que los elementos se comporten de manera predecible cuando el diseño de la página debe adaptarse a diferentes tamaños de pantalla y diferentes dispositivos de visualización.

  • Usando grid:

    • Grid está basado en cuadrículas, filas y columnas, lo que facilita el diseño de páginas web sin tener que usar elementos flotantes y posicionamiento.

Diseño Adaptable

El diseño web adaptable (responsive) consiste en crear páginas web que se vean bien en todos los dispositivos.

Un diseño adaptable se ajustará automaticamente a diferentes tamaños de pantalla y vistas.
